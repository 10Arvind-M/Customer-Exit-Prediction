#Customer Exit Prediction  

## Overview  
This competition focuses on predicting whether a customer of a financial institution will exit (churn) or continue their relationship with the organization. The task involves learning patterns from historical customer data that includes financial, demographic, and behavioral attributes to determine the likelihood of customer attrition.

## Problem Statement  
The objective is to build a classification model that predicts the exit status of each customer in the test dataset based on patterns learned from the training data.

## Target Variable  
The target variable represents whether a customer will exit or stay:  

- `1` – Customer exits  
- `0` – Customer stays  

## Evaluation Metric  
Submissions are evaluated using the **F1 Score**, which balances precision and recall for the binary classification task.
